Habilidades Técnicas

Python (Desarrollo Backend)

Uso de Python como lenguaje principal para el desarrollo de aplicaciones backend, implementando lógica de negocio, validaciones y manipulación de datos.

Experiencia trabajando con estructuras de control, funciones, módulos y organización del código en proyectos estructurados dentro del entorno Visual Studio Code.

Django Framework

Desarrollo de aplicaciones web dinámicas utilizando Django bajo el patrón Modelo – Vista – Template (MVT).

Implementación de modelos de base de datos, vistas dinámicas, formularios con validación automática y renderizado de información desde el servidor hacia la interfaz web.

Organización modular del proyecto siguiendo buenas prácticas del framework.

Bases de Datos y SQL

Diseño de bases de datos relacionales mediante creación de tablas, relaciones entre entidades, claves primarias y claves foráneas.

Desarrollo de consultas utilizando SELECT, JOIN, WHERE, GROUP BY y funciones agregadas para análisis y recuperación eficiente de información.

Integración de bases de datos dentro de aplicaciones web utilizando Django ORM.

Lenguaje C

Uso del lenguaje C para comprender el funcionamiento interno de los programas, manejo de memoria, estructuras de control y desarrollo de algoritmos eficientes.

Aplicación de conceptos fundamentales como punteros, estructuras y funciones modulares para resolver problemas computacionales.

Bootstrap (Diseño Frontend Profesional)

Construcción de interfaces web modernas utilizando Bootstrap para desarrollar layouts responsive, formularios estilizados, navegación estructurada y componentes reutilizables como cards y grids.

Integración de Bootstrap dentro de proyectos Django para lograr interfaces claras, accesibles y profesionales.

Visual Studio Code y Herramientas de Desarrollo

Uso de Visual Studio Code como entorno principal para el desarrollo de aplicaciones en Python, Django, SQL y C.

Organización de proyectos mediante estructura modular de carpetas, control de versiones con Git y publicación de proyectos en GitHub.
