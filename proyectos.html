Portfolio Web Profesional (HTML + CSS + GitHub Pages)

Desarrollo completo de un sitio web personal profesional utilizando Visual Studio Code como entorno principal de trabajo. El proyecto fue diseñado aplicando buenas prácticas de estructuración HTML semántica, organización modular del contenido y separación correcta entre estructura y estilos mediante CSS externo.

Se implementó un diseño responsive adaptable a distintos tamaños de pantalla, navegación entre múltiples vistas (Inicio, Sobre mí, Formación, Habilidades y Proyectos) y componentes reutilizables como cards, navbar y footer.

El proyecto fue publicado utilizando GitHub Pages, permitiendo su acceso público como portfolio técnico profesional.

Tecnologías utilizadas: HTML, CSS, Git, GitHub, Visual Studio Code

Base de Datos Académica en SQL

Diseño e implementación de un modelo relacional completo utilizando SQL, incluyendo creación de tablas, definición de claves primarias y foráneas, normalización de datos y verificación de integridad referencial.

Se desarrollaron scripts SQL para validar la estructura de la base de datos, permitiendo analizar la consistencia entre entidades y detectar errores de relación entre tablas.

Además, se realizaron consultas complejas utilizando SELECT, JOIN, WHERE, GROUP BY y funciones agregadas para obtener información relevante del sistema.

El proyecto fue desarrollado íntegramente desde Visual Studio Code utilizando scripts organizados por módulos.

Tecnologías utilizadas: SQL, Modelado relacional, Consultas avanzadas, Visual Studio Code

Aplicación Web con Django y Bootstrap

Desarrollo de una aplicación web dinámica utilizando Django como framework backend principal, aplicando el patrón de arquitectura MVT (Model – View – Template).

Se implementaron modelos de base de datos, vistas dinámicas y plantillas HTML integradas con Bootstrap para lograr una interfaz moderna y responsive.

La aplicación permite gestionar información estructurada mediante formularios web, validaciones automáticas del lado del servidor y renderizado dinámico de datos.

Bootstrap fue utilizado para construir layouts profesionales, cards, formularios estilizados y navegación adaptable a distintos dispositivos.

Todo el proyecto fue desarrollado utilizando Visual Studio Code como entorno principal.

Tecnologías utilizadas: Django, Python, Bootstrap, HTML, CSS, Visual Studio Code

Sistema de Reservas Hoteleras (Pascal)

Implementación de un sistema académico de gestión de reservas hoteleras utilizando Pascal, enfocado en el uso de estructuras dinámicas como listas enlazadas y registros.

El sistema permite registrar huéspedes, validar disponibilidad de habitaciones, almacenar información estructurada y recorrer dinámicamente la estructura de datos para generar reportes.

Se aplicaron técnicas de validación de datos, modularización mediante procedimientos y funciones, y manipulación eficiente de memoria dinámica.

Tecnologías utilizadas: Pascal, estructuras dinámicas, lógica estructurada

Estructuras de Datos en Java

Desarrollo de múltiples implementaciones de estructuras de datos clásicas utilizando Java, incluyendo listas enlazadas, pilas, colas simples y colas dobles.

Se aplicaron conceptos fundamentales de programación orientada a objetos como encapsulamiento, herencia, abstracción e implementación de tipos de datos abstractos (TDA).

También se desarrollaron algoritmos recursivos para recorrido y procesamiento de estructuras dinámicas.

Tecnologías utilizadas: Java, POO, estructuras dinámicas, recursividad

Prácticas de Arquitectura de Computadoras (8086)

Resolución de ejercicios prácticos en lenguaje ensamblador 8086 enfocados en manejo de pila, direccionamiento de memoria, subrutinas recursivas y paso de parámetros por valor y por referencia.

Se trabajó con simuladores académicos para analizar el comportamiento interno del procesador y comprender el funcionamiento de registros, flags y stack.

Estas prácticas permitieron comprender en profundidad la relación entre software y hardware.

Tecnologías utilizadas: Assembly 8086, arquitectura Von Neumann, subrutinas
